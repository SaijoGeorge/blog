View Google's Cache for any Link

Need help accessing a website that's down or changed? This bookmarklet offers a one-click solution to retrieve Google's cached version of any webpage instantly. Bypass downtime and view earlier versions of sites effortlessly, ensuring you always have access to the information you need. Drag the bookmarklet to your bookmarks bar, and enjoy uninterrupted web navigation, accessing cached pages with just a click.

Google officially confirmed on 2 February 2024 that they had removed the cache link in SERPs. For SEOs, this was a handy feature for quickly checking how a page appears in Google, if a website uses cloaking, etc. Fortunately, there's a workaround to access the cached URL directly; the bookmarklet leverages this. Hopefully, Google won't remove that.


Google Cache Link

Installation Instructions:

Drag the button above to your bookmarks bar. If you don't see the bookmarks bar, you might need to enable it in your browser settings.

How To Use:

  • When you're on the webpage you wish to find a cached version of.
  • Click the Bookmarklet "Google Cache Link" from you browser bookmarks bar.
  • A new tab will open with Google's cache version of that page.

Some Limitations:

  • If Google doesn't have a cached version of that page then the bookmarklet will not work. In that case you can try the Wayback Machine.
